## 概要

チームでオリジナルの紙芝居を作り、全員の前で発表するイベント。絵を描く人・ストーリーを考える人・語り手など役割分担が自然と生まれ、子供のイラストを大人が語るなど世代をまたいだコラボが面白さを生む。

## タイムライン

| 時間 | 内容 |
|------|------|
| 0:00〜0:10 | 集合・説明・チーム分け（5チーム×6人） |
| 0:10〜0:20 | テーマくじ引き・チーム内役割決め |
| 0:20〜1:00 | 制作タイム（ストーリー考案・絵を描く・セリフ練習） |
| 1:00〜1:45 | 発表本番（各チーム5〜8分） |
| 1:45〜2:00 | 表彰・全作品展示・記念撮影 |

## 準備・持ち物リスト

- A4用紙またはB5用紙 × 各チーム10〜15枚（計50〜75枚）
- マジック・色鉛筆・クレヨン × チーム数セット（100円ショップで1〜2セット）
- 割り箸またはストロー（紙芝居の持ち手用）× 各チーム数本
- テープまたは糊（紙を割り箸に貼る用）× 数個
- 発表台（段ボールを立てて紙芝居スタンドに）× チーム数分

## テーマくじ内容例

| テーマ | 内容 |
|--------|------|
| ヒーロー | 弱虫な主人公がみんなを助ける物語 |
| 食べ物 | 給食室のパンが消えた！犯人を探せ！ |
| 動物 | 森の動物たちが力を合わせて冬を乗り越える話 |
| 未来 | 2100年の地球でロボットと人間が友達になる話 |
| 地域 | この町の昔話・伝説をアレンジして描く |

## 審査基準・表彰

| 賞 | 内容 |
|----|------|
| 最優秀作品賞 | 総合的に最も楽しかった作品（参加者投票） |
| ベスト絵師賞 | 最も上手・かわいい・印象的な絵 |
| ベストストーリー賞 | 最もドキドキ・感動できた展開 |
| 語り手賞 | 最も引き込まれた語りをした人 |

## ミニゲーム・盛り上げアイデア

- **途中交換制**: 制作の途中で絵担当と文章担当を入れ替えてカオスな展開を楽しむ
- **効果音担当**: 発表時に他のチームが「ドドドン！」など効果音をリアルタイムで入れる
- **続編リクエスト**: 発表後に観客が「続きを聞きたい！」と手を挙げたら即興で続編を語る
- **子供作家賞**: 子供が描いたイラストの中で最も印象的なものを特別表彰

## 運営のポイント

- 制作前に「短くてOK・下手な絵でOK」を強調することで参加障壁を下げる
- 枚数の目安は「4〜8枚」と伝えるとまとまりやすい（多すぎると時間が足りなくなる）
- 語り手は子供でも大人でも複数人でも自由にしてよいと伝えるとチームの個性が出る
- 完成した紙芝居は壁に展示して会の後も見られるようにすると達成感が持続する
