## 概要

「こんな○○は嫌だ！」「このイラストに一言」など親しみやすいお題で笑いを競う大喜利大会。子供の天然ボケと大人のひねりが交差し、世代を問わず爆笑が生まれる。笑いを共有することで場の温度が一気に上がる。

## タイムライン

| 時間 | 内容 |
|------|------|
| 0:00〜0:10 | 集合・ルール説明・チーム分け（5チーム×6人） |
| 0:10〜0:20 | 練習ラウンド（低難度のお題で全員が1回答える） |
| 0:20〜1:10 | 本番（5〜6ラウンド実施、各ラウンドで得票を競う） |
| 1:10〜1:20 | 総合得点発表・MVP表彰 |
| 1:20〜1:30 | 名言まとめ発表・記念撮影 |

## 準備・持ち物リスト

- お題カード（A5紙に手書き）× 20〜30枚
- 回答用紙（ホワイトボードまたは紙）× チーム数分
- マーカー × チーム数本
- 投票用シール（丸シール）× 人数×3枚（100円ショップ）
- 得点ボード（段ボール＋マーカー）× 1枚

## お題例

| カテゴリ | お題例 |
|---------|--------|
| こんな〇〇はいやだ | 「こんな運動会はいやだ」「こんな給食はいやだ」 |
| 一言ボケ | 手書きのシュールなイラストに一言コメントをつける |
| なぜなぜ問答 | 「なぜカラスは黒いの？」（子供らしい答えが天才的になりやすい） |
| 大人気ない選手権 | 「大人気なくやってしまった行動」を告白し合う |
| 逆説 | 「失敗して良かったこと」を無理やり言う |

## ルール

1. 各チームは制限時間2分以内に回答を1つ書く
2. 全チームの回答を司会が読み上げる
3. 参加者全員が「一番面白いと思う回答」にシールを貼る
4. 得票数が最も多かったチームが1ポイント獲得

## ミニゲーム・盛り上げアイデア

- **子供専用ラウンド**: 子供だけが答え、大人が投票するラウンドを1回設ける
- **即興コント化**: 大喜利の名回答を題材に30秒の即興コントを演じてもらう
- **殿堂入り制度**: 特に面白かった回答は「名言ボード」に書き留めて壁に貼る
- **お題リクエスト**: 参加者が紙にお題を書いて投票箱に入れ、抽選で使うシステム

## 運営のポイント

- お題は「子供が答えやすいもの」と「大人がひねれるもの」を半々で用意するとバランスが良い
- 司会者の「読み上げ方・リアクション」が場の温度を決めるため、担当は盛り上げ上手な人を選ぶ
- 「笑えなかった回答」を茶化さないよう「どんな答えも拍手」のルールを最初に設定
- 子供の天然な回答は大人が予想もしない面白さがあるため、積極的に拾って称える
