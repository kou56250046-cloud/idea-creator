## 概要

参加者自身がすごろくの「コマ」になる体験型すごろく。じゃんけんに勝った数だけ前に進み、止まったマスに書かれた指令（お題）を実行する。指令の面白さと逆転要素が笑いを生み、何度やっても飽きない。

## タイムライン

| 時間 | 内容 |
|------|------|
| 0:00〜0:10 | マスに指令カードを並べる・ルール説明 |
| 0:10〜0:35 | 人間すごろく本番（全員が一斉にプレイ） |
| 0:35〜0:42 | ゴール者から順に「最も面白かったマス」を発表 |
| 0:42〜0:45 | 記念撮影 |

## 準備・持ち物リスト

- 床に並べる「マスカード」（A4紙を半分に切ったもの）× 20〜25枚（マス番号と指令を書く）
- 養生テープ（床にマスカードを固定）× 1巻
- スタート・ゴールの看板 × 各1枚（チラシ裏でOK）

## マスの指令例

| マス | 指令内容 |
|------|---------|
| 通常マス | 特になし・通過 |
| 自己紹介マス | 「好きな食べ物」を全員に宣言 |
| 逆戻りマス | 3マス後退（じゃんけん負け扱い） |
| お笑いマス | 30秒以内に全員を笑わせる一言を言う |
| ダッシュマス | 次のじゃんけん勝利で2マス進める |
| チーム戦マス | 最寄りの2人とグループじゃんけん（勝者だけ進める） |
| ワープマス | 任意の人のマスに瞬間移動 |
| 全員じゃんけん | 全員とじゃんけんして、最多勝利数だけ進む |
| 休憩マス | 1ターン休み（でも全員に好きなことを話せる） |
| ゴール手前マス | ゴールには3連勝しないと入れない |

## ルール

1. 参加者全員がスタートに立つ
2. 隣にいる人とじゃんけんをして、勝ったら1マス前進
3. 止まったマスの指令を実行してから次のじゃんけんへ
4. 最初にゴールに到達した人の勝利（全員ゴールも目標に）

## ミニゲーム・盛り上げアイデア

- **マスリクエスト**: 後半に参加者からのリクエストで「新しい指令マス」を即席で追加する
- **指令達成投票**: 自己紹介マス・お笑いマスなどで一番盛り上がった人に「指令賞」を設ける
- **速報実況**: 主催者が「○○さんが△マスでお笑いマスにかかりました！」とリアルタイム実況
- **大逆転マス**: コース終盤だけに「全員ゴールに戻す」カードを1枚仕込んでおくサプライズ

## 運営のポイント

- マスの間隔は30〜50cm程度で一直線またはくねくねしたコースに並べると見栄えが良い
- 指令マスは「誰が止まっても楽しめる」内容にし、苦手な人を追い詰めるものは避ける
- 人数が多い場合（20人）はコースを2本用意してチームごとに分けると進行がスムーズ
- 「ゴールした人は応援団に回る」ルールにすると最後まで全員が参加できる雰囲気になる
