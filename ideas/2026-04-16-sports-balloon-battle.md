## 概要

足首に風船を結び、相手の風船を踏んで割りながら自分の風船を守る風船バトル。狭いエリアで全員が同時に動き回るため、見ている側も爆笑の連続。子供は素早さで、大人は重さで戦略が変わるフラットな競技。

## タイムライン

| 時間 | 内容 |
|------|------|
| 0:00〜0:15 | 集合・ルール説明・風船を足首に結ぶ準備 |
| 0:15〜0:30 | 個人戦（全員参加・最後まで残った人を表彰） |
| 0:30〜0:45 | チーム戦（5チーム×6人・自チームの生存人数で競う） |
| 0:45〜0:55 | 休憩・次の風船を準備 |
| 0:55〜1:20 | 2本縛り版・ペア縛り版などアレンジバトル |
| 1:20〜1:30 | 最終決戦・表彰・記念撮影 |

## 準備・持ち物リスト

- 風船（丸型、直径20cm以上のもの）× 100〜120個（チーム色別に5色）
- 100円ショップの風船セット（5色×各20個入り）× 3〜4袋（300〜400円）
- ひもまたはリボン（風船を足首に結ぶ用）× 人数分
- 予備ポンプまたは息で膨らませる（会場でみんなで膨らませると盛り上がる準備になる）

## ルール

1. 自分の足首に風船を1本結ぶ（長さ30〜40cm程度）
2. 合図とともに全員が動き始め、相手の風船を踏んで割る
3. 自分の風船が割られたらエリア外へ
4. 最後まで風船が割れずに残った人（チーム）の勝利

## アレンジルール

| バリエーション | ルール |
|-------------|--------|
| **2本縛り** | 両足に1本ずつ結び、2本両方割られるまで戦える |
| **チーム制** | 同チームメンバーとは協力して敵チームの風船を狙う |
| **ペアリンク** | 2人が手をつないだ状態で戦い、片方が割られたら2人とも脱落 |
| **大人ハンデ** | 大人は片足縛り（片足でケンケンしながら戦う） |

## ミニゲーム・盛り上げアイデア

- **準備で盛り上がる**: 会場で全員一斉に風船を膨らます時間がアイスブレイクになる
- **サバイバーインタビュー**: 優勝した最後の1人に「どんな戦略でしたか？」とインタビュー
- **割った数コンテスト**: 勝敗だけでなく「最多割り記録」を個人賞として表彰
- **演技派賞**: 割られた瞬間の「リアクション」が最も面白かった人を特別表彰

## 運営のポイント

- 風船は膨らませすぎると割れやすく、小さすぎると踏みにくいため「ゲンコツより少し大きめ」が適切
- バトルエリアは参加人数に応じて設定（30人なら10m×10m程度）、広すぎると逃げ続けるだけになる
- 小さい子供が大人に踏まれるリスクがあるため「子供エリア内の大人は立ち止まって戦う」ルールを設けると安全
- 割れた風船のゴムの散乱は誤飲の危険があるため、すぐに回収係を設ける
