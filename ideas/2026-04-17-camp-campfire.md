## 概要

キャンプ場または焚き火が許可された屋外スペースで大きな火を囲み、歌・ゲーム・語り合いを楽しむ夜のイベント。「炎を囲む」という原始的な体験が時間をゆっくりと感じさせ、自然と本音で話せる雰囲気を作り出す。子供の頃に誰もが憧れた焚き火の記憶が大人の心も解きほぐす。

## タイムライン

| 時間 | 内容 |
|------|------|
| 0:00〜0:20 | 集合・薪集め・火おこし（全員参加） |
| 0:20〜0:50 | ファイヤーゲーム（火を囲んでできるゲーム） |
| 0:50〜1:30 | キャンプソング・合唱タイム |
| 1:30〜2:00 | マシュマロ焼き・スモア体験 |
| 2:00〜2:40 | 焚き火トーク（少人数に分かれて語り合う） |
| 2:40〜3:00 | 火の消火・後片付け・解散 |

## 準備・持ち物リスト

- 薪（ホームセンター、またはキャンプ場で調達）× 1束（500〜800円）
- 着火剤または新聞紙 × 適量
- バケツ・水（消火用）× 1〜2個
- マシュマロ × 1〜2袋（200〜300円）
- 竹串または長い棒 × 人数分
- ビスケット・板チョコ（スモア用）× 適量
- ランタンまたは懐中電灯 × 数個（安全確保用）

## ファイヤーゲーム一覧

| ゲーム | 内容 |
|--------|------|
| 火の番リレー | 火が消えないよう交代で薪をくべる「火守り」役 |
| 影絵遊び | 炎の光でできた影を使って手影絵を見せ合う |
| 焚き火ストーリー | 順番に1文ずつ繋げてオリジナルの怪談・冒険話を作る |
| 炎の色クイズ | 薪の種類や燃え具合によって炎の色が変わる理由を考える |

|------|------|
| 今日の日はさようなら | しっとりとした感動的な締め曲 |
| 燃えろよ燃えろ | 焚き火にぴったりの盛り上がり曲 |
| 遠き山に日は落ちて | 夜の静けさを演出するクラシック |
| ビリーブ | 合唱しやすく子供も大人も歌える |

## ミニゲーム・盛り上げアイデア

- **焚き火トークテーマカード**: 「一番怖かった体験」「生まれ変わったら何になりたい」などを記したカードをくじ引きで引いて話す
- **炎の芸術**: 薪を美しく組む「炎のアート」を競う組み方コンテスト
- **スモアアレンジコンテスト**: マシュマロの焼き加減・挟み方を各自が工夫して一番おいしいスモアを競う
- **消灯セレモニー**: 最後に全員で「今日の感謝を一言ずつ言いながら火を消す」締めの儀式

## 運営のポイント

- 火気使用は事前に会場への確認・許可を必ず取る（消防法・公園管理条例への対応）
- 焚き火から離れない「火守り大人スタッフ」を常時1名以上配置する
- 子供が炎に近づきすぎないよう「安全ゾーン（1m以内立入禁止）」をロープで区画する
- 終了後は完全消火を確認し、焚き火の跡地を清掃してから帰る
