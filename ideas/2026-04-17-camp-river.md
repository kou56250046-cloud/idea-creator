## 概要

安全な川辺に集まり、石積み（ケルン）・川魚観察・水切りゲームなど川の自然を満喫するリトリートイベント。スマホを置いて川の音と触感だけに集中することで、日常の喧騒から切り離されたリラックス体験ができる。子供は水遊びを、大人は静かな石積み瞑想を楽しめる構成。

## タイムライン

| 時間 | 内容 |
|------|------|
| 0:00〜0:15 | 集合・安全確認・川辺のルール説明 |
| 0:15〜0:50 | 石積みチャレンジ（どれだけ高く積めるか） |
| 0:50〜1:20 | 水切りゲーム（石が何回跳ねるか競争） |
| 1:20〜1:50 | 川魚・生き物観察（網があれば採取体験も） |
| 1:50〜2:30 | 自由時間（水遊び・読書・昼寝・写真撮影） |
| 2:30〜3:00 | 片付け・まとめトーク「今日一番印象に残った瞬間」 |

## 準備・持ち物リスト

- レジャーシート（川辺で座る用）× 3〜4枚
- タオル × 人数分（各自持参）
- 着替え・防水サンダル（各自持参・事前告知）
- 小型の虫取り網（任意）× 2〜3本
- 救急セット（消毒液・絆創膏）× 1セット
- ゴミ袋（川辺清掃兼用）× 数枚
- 飲料水 × 人数分（熱中症対策）

## 石積みチャレンジルール

| ルール | 内容 |
|-------|------|
| 基本ルール | 川辺の石のみ使用、道具・接着剤禁止 |
| タイム | 15分間で最も高く積む |
| 計測 | 積み上げた高さをcmで計測 |
| ボーナス | 「最も芸術的」「最もバランスが危うい」などの追加賞 |

## ミニゲーム・盛り上げアイデア

- **石に名前をつけよう**: 気に入った石に名前・ニックネームを付けて「石の紹介タイム」を行う
- **水音傾聴瞑想**: 1分間、川の音だけに集中して「聞こえた音の種類を数える」マインドフルネス
- **ゴミ拾いスコアラリー**: 川辺のゴミを拾い「珍しいゴミ」「多かった種類」を発表する環境教育
- **水切り大会**: 石が何回跳ねたかを数え、最多回数チャンピオンを表彰する

## 運営のポイント

- 川は必ず水深・流速を事前確認し、増水時・増水後24時間以内は中止する
- 子供は必ず大人と水辺に行き、単独での川への入水は禁止する
- 川魚・生き物を採取した場合は観察後に必ず元の場所に戻す
- 熱中症対策として日陰エリアの確保と定期的な水分補給を促す
