## 概要

紅葉の時期に合わせて近くの公園・山・里山を歩きながら秋の自然を満喫するウォーキングイベント。歩きながら自然に隣の人との会話が生まれ、道中の発見・驚きがそのまま共有体験になる。ゴールに小さなお楽しみを用意することで全員が最後まで歩き切れる。

## タイムライン

| 時間 | 内容 |
|------|------|
| 0:00〜0:15 | 集合・準備体操・コース説明 |
| 0:15〜1:15 | 前半ウォーキング（自然観察・写真撮影） |
| 1:15〜1:30 | 中間休憩・発見物シェアタイム |
| 1:30〜2:15 | 後半ウォーキング（落ち葉・木の実収集） |
| 2:15〜2:45 | ゴール地点でお楽しみタイム（持ち寄りおやつ） |
| 2:45〜3:00 | 記念撮影・振り返り |

## 準備・持ち物リスト

- 観察記録カード（A6用紙・手書きで作成）× 人数分
- ペン × 人数分（観察記録用）
- 持ち寄りおやつ（各自100〜200円程度）× 人数分
- 敷物またはレジャーシート（ゴール地点用）× 2〜3枚
- 救急セット（絆創膏・消毒液）× 1セット
- ゴミ袋 × 数枚

## 観察チャレンジカード

| 課題 | 内容 |
|------|------|
| 赤い葉を5種類 | 色の違いを観察して記録 |
| 松ぼっくり・ドングリ | 形の違いを記録・拾う |
| 鳥の声を3種類 | 声だけで何の鳥か予測 |
| 最も美しい紅葉スポット | スマホで撮影してゴールで発表 |
| 秋の匂いを3つ | どんな匂いか言葉で表現 |

## ミニゲーム・盛り上げアイデア

- **フォトコンテスト**: 道中で撮った「秋の一枚」をゴール地点で発表・投票
- **落ち葉アート**: 収集した落ち葉をシートの上に並べて絵や模様を作る
- **秋の色ランキング**: 全員が「今日見た一番好きな色」を言葉で発表して共有
- **自然物ビンゴ**: 観察記録カードをビンゴ形式にして「全部見つけた人」を表彰

## 運営のポイント

- コースの距離は2〜4kmが目安（子供の足で歩き切れる距離）
- 紅葉の見頃は地域・標高によって大きく異なるため、事前に現地確認を行う
- 足元が滑りやすい場所（濡れた落ち葉・坂道）では全員に注意を促す
- 子供が疲れた場合の「リタイア手段（送迎担当の大人を1人用意）」を事前に設定しておく
