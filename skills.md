# イベント企画スキルリファレンス

このファイルは AI がイベントアイデアを生成する際の品質・多様性を高めるための参照データです。
アイデア生成時は以下のフレームワーク・原則・データベースを活用してください。

---

## 1. 発想フレームワーク: SCAMPER法

10個のアイデアを生成する際は、以下7視点を意識的に使い分けて多角的に展開する。

| 視点 | 意味 | 地域イベントへの応用例 |
|------|------|----------------------|
| **S** Substitute（代替） | 定番の要素を別のものに置き換える | 運動会の「玉入れ」を「水風船入れ」に |
| **C** Combine（組合せ） | 2つの異なるイベントを融合する | 料理教室 × クイズ大会 |
| **A** Adapt（適応） | 他業界の形式をコミュニティに転用 | 企業の研修ゲームを地域向けにアレンジ |
| **M** Modify/Magnify（変更） | 規模・頻度・強度・テーマを変える | 普通の写真撮影会をテーマ縛り（昭和レトロ）に |
| **P** Put to other uses（転用） | 日常の場所・道具をイベントに使う | 公民館の調理室で料理対決 |
| **E** Eliminate（削除） | 余計な複雑さを取り除く | 道具ゼロ・移動ゼロで楽しめるゲームだけに絞る |
| **R** Reverse/Rearrange（逆転） | 役割・順序・常識を逆転させる | 子供が先生、高齢者が生徒になる料理教室 |

## 2. イベントカテゴリ（6種類）

アイデアは以下6カテゴリに分類し、偏りなく生成する。

| カテゴリ | アイコン | 特徴 | 向いている対象 |
|---------|---------|------|--------------|
| 体験・ワークショップ型 | 🛠 | 何かを作る・学ぶ | 全世代（テーマで調整） |
| 競技・ゲーム型 | 🏆 | チーム対抗・競い合い | 子供〜大人（ハンデ調整必要） |
| 食・飲食型 | 🍽 | 食を中心にした交流 | 全世代（定番かつ強力） |
| 文化・芸術型 | 🎨 | 音楽・演劇・伝統・展示 | 高齢者・子供・文化好き |
| 自然・アウトドア型 | 🌳 | 屋外活動・散策・自然体験 | 体力のある世代向き（工夫次第で全員） |
| 奉仕・協力型 | 🤝 | 地域への貢献・共同作業 | 大人・高齢者・やりがい重視層 |

## 3. 親睦を深める心理的原則

以下の原則を満たすイベントほど、参加後の満足度と関係深化が高まる。アイデア生成時はこれらを意識する。

### 🤝 共同作業効果
一緒に何かを作ったり、協力して目標を達成すると、参加者同士の親密度が大幅に上がる。
→ 「チームで作る」「全員で達成する」要素を入れる

### 😄 笑いの力
一緒に笑った経験は、言葉より強く関係を結ぶ。
→ 失敗が笑えるゲーム・ちょっと恥ずかしい参加要素を入れる

### 💬 開示の返報性
自己開示すると相手も開示しやすくなり、仲が深まる。
→ 自己紹介・思い出・好みを話す機会を設ける

### 🎯 ピーク・エンド則
人の記憶に残るのは「最も感情が動いた瞬間」と「終わり方」。
→ クライマックスとなるサプライズや表彰を用意し、締め方を丁寧にする

### 👥 社会的証明
他者の参加・熱中が自分の参加意欲を高める。
→ 観客が盛り上がれる仕掛けを入れる（声援・リアクション・投票）

### 🔄 フロー理論
難しすぎず、簡単すぎない「ちょうどいい挑戦」が没入感を生む。
→ レベル別の役割・ハンデ設定で全員が適度に頑張れるようにする

## 4. 全世代が楽しめる設計のコツ

### ハンデ・役割の多様化
- 競技者だけでなく「応援者」「審判」「記録係」「MC」など役割を用意する
- 体力・年齢によるハンデを明示的に設定する
- 座ったままでも参加できる方法を必ず用意する

### 子供が輝ける瞬間を作る
- 大人が苦手で子供が得意な要素（柔軟性、勘、瞬発力）を含める
- 子供の活躍を大人が自然と認める場面を設ける

### 高齢者が活躍できる瞬間を作る
- 経験・知恵・昔の記憶が活かせる問題や場面を作る
- 無理な動きが不要な役割を用意する

## 5. 評価基準（スコアリングルール）

### 実現性スコア（1〜5点）

| スコア | 基準 |
|--------|------|
| 5 | 準備ほぼ不要。当日その場で即実施できる |
| 4 | 1週間前から動けば十分。特別な機材・許可不要 |
| 3 | 2〜4週間の準備が必要。材料調達や事前告知が必要 |
| 2 | 専門知識・高価な機材・外部業者が必要 |
| 1 | 許認可取得・大規模な設営・専門スタッフが必要 |

### 新規性スコア（1〜5点）

| スコア | 基準 |
|--------|------|
| 5 | ほぼ誰も見たことがない斬新な組み合わせ・発想 |
| 4 | 珍しいが類似例は存在する。地域では初めてレベル |
| 3 | よくあるイベントだが、独自のアレンジが加わっている |
| 2 | 一般的だが地域では久しぶり・初めてかもしれない |
| 1 | 定番中の定番。目新しさは特にない |

## 6. ミニゲームバンク（すぐ使えるネタ集）

### 全員参加系（準備不要）
- **〇×クイズ**: 問題に合わせて立つ・座るで回答。全員参加しやすい
- **人間知恵の輪**: 10人程度で手をつなぎ、絡まりをほどく協力ゲーム
- **ジェスチャー伝言ゲーム**: お題を声なしで伝える。笑いが生まれる
- **あいうえお作文対決**: テーマの言葉で即興あいうえお作文。チーム対抗
- **チームビルディングじゃんけん**: トーナメントじゃんけん＋敗者が勝者を応援するシステム

### 知識・記憶系
- **地域写真クイズ**: 昔の地域の写真や変化前の場所を当てる
- **食材ブラインドテスト**: 目隠しで食材・食品を当てる
- **昔の値段クイズ**: 「〇〇年前にラーメンはいくら？」

### 体を使う系（ハンデ調整可能）
- **風船バレー**: ラリー回数を競う。全世代参加可
- **新聞紙タワー**: チームで新聞紙とテープだけで高いタワーを作る
- **ペットボトルボーリング**: 空のペットボトルで作る即席ボーリング

### 表現・創造系
- **即興CM作り**: 商品を決めて30秒のCMをチームで考えて発表
- **お題ドローイング**: お題を絵で描いて当ててもらう（Pictionary形式）
- **寸劇チャレンジ**: お題のシーンを即興で演じる

## 7. 重複排除の観点

以下の観点でアイデアの独自性を確認し、類似するものは統合・差し替えする。

- **活動の種類**が同じ（例：料理系が3個はNG → 1〜2個に絞る）
- **場所・設定**が同じ（例：全員が公園はNG → 屋内/屋外/場所を分散）
- **主役となる世代**が偏っている（例：子供向けばかりはNG → 世代を分散）
- **予算帯**が偏っている（高コスト・低コストをバランスよく混ぜる）
- **体力要求度**が偏っている（激しい系・ゆるい系を混ぜる）
